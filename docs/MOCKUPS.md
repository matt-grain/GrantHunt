## Tech Stack for Phase 4-5

**Option A: Python + HTMX (Minimal)**
- FastAPI backend (reuse existing CLI logic)
- HTMX for interactivity
- Tailwind CSS
- SQLite (same DB)

**Option B: Next.js (Richer UX)**
- Next.js App Router
- React Server Components
- shadcn/ui
- SQLite via Drizzle ORM

**Option C: Local Electron App**
- Electron wrapper
- React frontend
- Direct SQLite access
- Works offline

**Recommendation:** Option A for MVP - least new code, reuses Python backend.

---

## Implementation Notes

- Dashboard reads from same `jobs.db` as CLI
- Actions trigger CLI commands under the hood
- Research/cover letter artifacts served from `applications/` folder
- Real-time updates via SSE or polling
